# Hardwood HQ

A front-office NBA analytics terminal: team builder, player grades, comparisons,
salary-cap tracking, and a fit engine that suggests roster additions ranked by
projected wins added.

> The player data in `src/App.jsx` is **synthetic sample data** for now.
> The spot marked `ROSTER_SOURCE` is where real data gets plugged in later.

## How to get it online (no coding required)

1. **Put it on GitHub.** Install GitHub Desktop, sign in (free account), and
   add this folder as a new repository. Click "Publish."
2. **Deploy with Vercel.** Sign in to Vercel with your GitHub account, click
   "Add New Project," pick this repository, and click "Deploy." Vercel installs
   everything and builds it for you. About a minute later you get a live web address.
3. That's it — your changes to GitHub auto-redeploy.

## Running it on your own computer (optional, needs the terminal)

```
npm install
npm run dev
```

Then open the address it prints (usually http://localhost:5173).
